# Prices

List currency price quotations in realtime.

Possible tokens are: XLM, XRP, BTC, ETH, CELO, BNB, MDA, HTR, ADA, AVAX, CHZ, SOL, MATIC, BRL, USD.

```js
const sdk = new CryptumSdk({
  environment: 'testnet',
  apiKey: 'YOUR-API-KEY'
})
// list for XLM
const prices = await sdk.prices.getPrices('XLM')
```
